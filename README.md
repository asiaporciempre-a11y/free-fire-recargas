# 🎮 Free Fire Recargas - Yape Bolivia

Página de venta de diamantes y membresías de Free Fire con integración de Yape para Bolivia.

## 📋 Características

- ✅ Interfaz moderna y atractiva
- ✅ Selector de paquetes de diamantes
- ✅ Integración con Yape para pagos
- ✅ Enlace automático a WhatsApp
- ✅ Responsive (funciona en móvil)

## 🚀 Desplegar en Netlify

1. Ve a [Netlify](https://netlify.com)
2. Conecta tu cuenta de GitHub
3. Selecciona este repositorio (`free-fire-recargas`)
4. Netlify desplegará automáticamente
5. Tu sitio estará en: `https://tudominio.netlify.app`

## ⚙️ Configuración

Para usar tu propio QR de Yape, edita el archivo `index.html` y reemplaza esta línea:

```javascript
const QR_IMAGE_URL = "https://tu-url-del-qr.com/qr-yape.png";
```

Con la URL real de tu QR alojado en:
- Imgur
- Imgbb
- Cloudinary
- O cualquier servicio de hosting de imágenes
